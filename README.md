# velocity_website
Responsive landing page built with **HTML, CSS, and JavaScript**

A clean and responsive landing page built with **HTML, CSS, and JavaScript**.  
This project demonstrates a professional one-page website structure — perfect for showcasing creative services or an agency profile.

Features
- Fixed navigation bar with smooth scrolling
- Responsive hero section with background image
- “What We Do” and “Services” blocks with cards
- Interactive **tab section** (switching content dynamically with JS)
- Fully responsive layout for all screen sizes

Technologies Used
- **HTML5** – semantic layout  
- **CSS** – styling, flexbox, media queries  
- **JavaScript (ES6)** – tab functionality and smooth scroll  
- **SVG Icons** – vector graphics for clean design  
- **Reset.css** – consistent base across browsers
